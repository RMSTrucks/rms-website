# RMS Truckers Website

Commercial trucking insurance website for RMS Insurance.

## Tech Stack

- **Framework:** Astro
- **Hosting:** Vercel (planned)
- **Domain:** rmstruckers.com

## Local Development

```bash
npm install
npm run dev
```

Site runs at `http://localhost:4321`

## Build

```bash
npm run build
```

Output goes to `dist/` folder.

## Deployment

1. Push to GitHub
2. Connect repo to Vercel
3. Auto-deploys on push to main

## Pages

- `/` - Homepage
- `/new-authorities` - Landing page for new authority holders
- `/blog/pricing-truth` - SEO article on pricing

## Content Updates

Edit `.astro` files in `src/pages/`. Changes auto-reload in dev mode.
